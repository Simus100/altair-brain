# Struttura del report di analisi

Scheletro del report analitico (dai project work): 

1. **Introduzione** — descrizione del dataset e obiettivi.
2. **Data cleaning** — mancanti, duplicati, outlier, correzioni.
3. **Analisi esplorativa** — distribuzioni e pattern.
4. **Analisi relazionale** — confronti tra variabili.
5. **Insight** — cosa abbiamo scoperto.
6. **Conclusioni** — implicazioni operative/di business.

Le colonne derivate si mappano sulle pagine delle dashboard (es. panoramica temporale; prezzi & ricavi; cancellazioni). Realizzazione con [[power-bi]] o Looker Studio; il filo conduttore e lo [[data-storytelling]] (hook emotivo -> dati -> insight).
