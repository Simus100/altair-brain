# Workflow di analisi dati (end-to-end)

Il metodo ripetibile per analizzare QUALSIASI dataset, ricavato dai project work reali. Sette fasi:

1. **Import e strutturazione** — importazione (Testo/CSV) e trasformazione in tabella.
2. **Controlli di qualita** — [[controlli-qualita-dati]]: duplicati, mancanti, outlier, coerenza.
3. **Feature engineering** — [[feature-engineering]]: colonne derivate per l'analisi.
4. **Analisi esplorativa** — [[analisi-esplorativa]]: statistiche descrittive, pivot, distribuzioni.
5. **Analisi relazionale** — confronti incrociati tra variabili (X vs Y).
6. **Insight** — trasformare i numeri in narrativa di business.
7. **Report** — [[struttura-report-analisi]] con [[data-storytelling]].

Le domande da porre lungo il percorso: [[framework-domande-analitiche]].
