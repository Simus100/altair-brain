- IMPLEMENTAZIONE


*creiamo un virtualenv dedicato:

% python -m venv mongodb-python

*verifichiamo presenza directory:

% ls -al

*attiviamo il virtualenv:

% source mongodb-python/bin/activate

*installare pymongo:

% pip install pymongo

*installiamo nel venv anche jupyter per avere un notebook:

% pip install jupyter

*avviamo il notebook:

% jupyter notebook
