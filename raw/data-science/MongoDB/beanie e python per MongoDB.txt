- INTRO A BEANIE


Beanie è un ODM (Object-Document Mapper) asincrono per MongoDB in Python.




!pip install beanie motor


- creo con compass il database chiamato "imdb_per_poveri" e una collection chiamata "movies". Copio anche l'indirizzo del server locale.

- Non in un jupyter ma in un file app.py nella cartella mongodb-python:


- importiamo le librerie che servono:


from pydantic import BaseModel, Field

from typing import List, Optional

from beanie import Document


import asyncio



from motor.motor_asyncio import AsyncIOMotorClient

from beanie import init_beanie



- Costruiamo un catalogo semplice di films:



class Genre(BaseModel):
   name: str

class Actor(BaseModel):
   name: str

class Movie(Document):
   title: str
   release_year: int
